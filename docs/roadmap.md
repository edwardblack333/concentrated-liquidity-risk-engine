# Roadmap

Near-term packaging goals:

- Add small public sample input files with documented schemas.
- Add schema checks for each local master dataset.
- Add dry-run examples for local rebuild and reporting scripts.
- Add a reproducible sample report build using synthetic or heavily reduced data.

Research/product goals:

- Separate external incremental refresh from local model rebuilds.
- Expand validation beyond in-sample diagnostics.
- Add pool-specific calibration workflows for any new pool.
- Improve WAIT/deploy threshold diagnostics without changing production thresholds casually.
- Improve public documentation of expected input schemas and sample-data rebuild steps.
